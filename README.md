  *Python,科研项目*

  快速存取记录系统（Quick Access Recorder, QAR）得到的原始飞行参数数据具有**高频、海量、异频、高维、相关性严重**等特性，无法直接充当机器学习预测模型训练的数据集。因此，我使用Pandas和Numpy库进行**数据清洗**，并用SPSS主成分分析进行**数据降维**，训练得到的决策树模型预测效果良好。

  + 决策树模型下测试集准确率达到98.55%
